# Teklini Auth Screens

Responsive sign-in / sign-up UI with a sliding panel and clean styles.

## Overview
This repository contains a simple frontend-only authentication screens demo built with HTML and CSS. It's a lightweight template you can use as a starting point for login and registration pages.

## Features
- Sliding panel to switch between Sign In and Sign Up views
- Responsive layout for mobile and desktop
- Social icons (Font Awesome) for visual demo
- Simple, clean styling using Poppins font

## Files
- `index.html` — main demo page
- `style.css` — styling for the UI

## Usage
1. Clone the repo:

```bash
git clone https://github.com/bmcouma/teklini-auth-screens.git
cd teklini-auth-screens
```

2. Open `index.html` in your browser to preview.

## Customize
- Replace the footer credit text in `index.html` if needed.
- Add your logo under `assets/` and update the footer markup.
